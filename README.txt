
Here are setup instructions for "benchmark_deep1B_gpu_cpu_only.py":

* You should already have installed CUDA version 11.0 or compatible (higher may we, we only tested on 11.0)
* We downloaded and installed the Anaconda distribution of python
* We created a python environment using python=3.8.5 and activated it
* Installed python dependencies using "faiss-gpu_requirements.txt" requirements file
* Edit the benchmark_deep1B_gpu_cpu_only.py with your local settings ( paths to various index and ground truth files. )
* Run "python benchmark_deep1B_gpu_cpu_only.py"
